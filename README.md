# SOVEREIGN

A reinforcement-learning environment for studying whether a militarily superior
agent learns that **invasion is a strategically dominated strategy** when the
model includes the costs that real coercive action incurs — legitimacy decay,
occupation drag, sanctions, insurgency hazard, and shifting third-party posture.

Three nations on a graph. The invader is a PPO agent. The defender is
rule-based. The neutral is a stochastic political process. Five ablation
regimes toggle the cost mechanisms independently to identify which one is
load-bearing.

**Headline finding:** legitimacy is the load-bearing mechanism. Without it,
the agent loses the ability to *end* the game through negotiation — aggression
becomes costly not because of any single penalty, but because it forecloses
the negotiated exit.

See `PREP.md` for a comprehensive walkthrough of the design, algorithms, and
results. See `Sovereign.pdf` for the rulebook.

## How to execute

All commands assume you are in the repository root. The project uses [`uv`](https://docs.astral.sh/uv/);
plain `pip` works just as well — substitute `uv run python` with `python` if you prefer.

### 1. Install

```bash
pip install -e ".[dev,train,ui]"
```

The three extras are independent:
- `dev` — pytest, ruff, black (test toolchain only)
- `train` — Stable-Baselines3, PyTorch, TensorBoard, matplotlib, pandas
- `ui` — FastAPI, uvicorn, pydantic

For the React frontend:
```bash
cd sovereign/ui/frontend && npm install
```

### 2. Run the tests

```bash
pytest
```

~40 tests covering pure-function mechanics (drift, hysteresis, combat, hazard,
reward decomposition) and env-level behaviour (action codec, observation
shape/bounds, the iter-9 settlement gate).

### 3. Train a single regime

```bash
uv run python -m sovereign.training.train_ppo --regime full --steps 1000000
```

Writes `final.zip`, `ckpts/`, `episodes.csv`, and `tb/` (TensorBoard logs) to
`results/ppo/full_rulebook9_s0/`.

Flags:
- `--regime {full, no_legitimacy, no_occupation_cost, no_neutral, baseline}`
- `--map {rulebook9, default9, frontier12}` (default `rulebook9`)
- `--steps INT` (default 1,000,000)
- `--n-envs INT` (default 4, vectorised)
- `--seed INT` (default 0)
- `--lr FLOAT` (default 3e-4)

### 4. Run the full ablation protocol

```bash
uv run python -m sovereign.experiments.ablations --steps 1000000 --eval-episodes 64
```

Trains one PPO model per regime, evaluates each across 64 deterministic
greedy episodes, writes `results/ablations/<regime>/eval.json` and
`results/ablations/summary.json`. Wall time ≈ 45–50 minutes on a 2024
MacBook M-series.

Quick smoke test (≈ 30 seconds total):
```bash
uv run python -m sovereign.experiments.ablations --quick
```

### 5. Parameter sweeps + plots

```bash
uv run python -m sovereign.experiments.parameter_sweep --steps 50000
uv run python -m sovereign.experiments.plots
```

Figures land in `results/figures/`.

### 6. Run the live demo

Replays a captured deterministic trace from `demo_traces/`. Three regimes
shipped:

```bash
uv run python -m sovereign.experiments.demo_episodes full
uv run python -m sovereign.experiments.demo_episodes baseline
uv run python -m sovereign.experiments.demo_episodes no_legitimacy
```

Pacing flags:
- `--delay FLOAT` — per-line delay in seconds (default 0.02)
- `--instant` — dump the trace immediately

To **re-capture** traces from a fresh training run:

```bash
mkdir -p demo_traces
uv run python -m sovereign.experiments.capture_episodes --regime full --episodes 3 --seed 0 > demo_traces/full.txt 2>&1
uv run python -m sovereign.experiments.capture_episodes --regime baseline --episodes 2 --seed 0 > demo_traces/baseline.txt 2>&1
uv run python -m sovereign.experiments.capture_episodes --regime no_legitimacy --episodes 1 --seed 0 > demo_traces/no_legitimacy.txt 2>&1
```

See `demo-script.md` for the full speaker notes.

### 7. Launch the UI (optional)

> **Note:** the UI is a work-in-progress. The backend serves the data and the
> frontend renders the basics, but several views are still stubs — we plan to
> finish it in a follow-up iteration. All headline results in `PREP.md` come
> from the CLI workflows above; the UI is not required to reproduce anything.

Backend (port 8000):
```bash
uv run python -m sovereign.ui.backend.serve
```

Frontend (port 5173) — in another shell:
```bash
cd sovereign/ui/frontend && npm run dev
```

The UI reads from `results/` directly. The *Compare* tab consumes
`results/ablations/summary.json`.

## Configuration

All coefficients, weights, and ablation flags live in
`sovereign/env/config.py`. To override from a YAML file:

```yaml
# experiment_a.yml
hazard:
  lam: 0.10
reward:
  occupation: 0.40
flags:
  legitimacy_enabled: true
```

```python
from sovereign.env.config import SovereignConfig
cfg = SovereignConfig.from_yaml("experiment_a.yml")
```

## Reading the results

After running the ablation protocol, `results/ablations/summary.json` looks
like:

```json
[
  {"regime": "full",           "settlement_rate": 1.00, "return_mean": 71.76, "mean_legitimacy": 0.47},
  {"regime": "no_legitimacy",  "settlement_rate": 0.00, "return_mean": 25.63, "mean_legitimacy": 1.00},
  {"regime": "no_occupation_cost", "settlement_rate": 0.98, "return_mean": 69.50, "mean_legitimacy": 0.54},
  {"regime": "no_neutral",     "settlement_rate": 1.00, "return_mean": 67.85, "mean_legitimacy": 0.56},
  {"regime": "baseline",       "settlement_rate": 0.00, "return_mean": 77.14, "mean_legitimacy": 1.00}
]
```

The three regimes where L can move all settle; the two where L is locked
at 1.0 time out. That's the load-bearing-legitimacy result.

## Maps

Three topologies ship registered:
- **`rulebook9`** — default. One home per nation + six contested territories. Matches `Sovereign.pdf` §3.3.
- **`default9`** — legacy simplified map: 3 invader-home + 3 defender-home + 3 contested.
- **`frontier12`** — longer-front variant for sweeps on graph structure.

Register your own:
```python
from sovereign.env.map import MapSpec, Territory, register_map, INVADER, DEFENDER, CONTESTED

@register_map("mymap")
def _mymap() -> MapSpec:
    return MapSpec("mymap", territories=(...), edges=(...))
```

## Inheritance

The mechanics layer is inspired by Meta FAIR's CICERO / Diplomacy line of
work — specifically the observation that durable cooperative outcomes
require agents that internalise the cost of unilateral action on third-party
trust. SOVEREIGN is a much smaller, more transparent setup (one learning
agent, two scripted opponents, six mechanism levers) designed to ask which
of those mechanisms, individually, is *necessary* for the dominated-strategy
result to emerge under PPO. See `RESEARCH.md` for the full argument and
`PREP.md` for the implementation walkthrough.
